#  Movie Recommendation System

##  Overview
This project is an  Movie Recommendation System built using collaborative filtering and matrix factorization techniques.

It suggests personalized movies to users based on their past ratings.

---

##  Features
-  Personalized movie recommendations  
-  Collaborative Filtering (User-Item)  
-  SVD (AI-based prediction)  
-  Data Visualization (ratings, popular movies)  
-  Interactive UI using Streamlit  

---

##  Tech Stack
- Python   
- Pandas & NumPy  
- Scikit-learn  
- Matplotlib & Seaborn  
- Streamlit  

---

##  Dataset
- MovieLens Dataset  
- Contains movies, ratings, and user interactions
